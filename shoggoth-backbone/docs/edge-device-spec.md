## 2. Software Stack (Pre-Installed)

### Operating System
- **ShoggothOS**: Ubuntu Server 24.04 LTS with custom kernel (6.12+).
  - CONFIG_DMA_SHARED_BUFFER=y
  - CONFIG_HUGETLBFS=y (1 GB huge pages)
  - CONFIG_CRYPTO_DEV_QAT=y (Intel QAT)
  - CONFIG_VSOCKETS=y (AF_HYPERV)
  - PREEMPT_RT patch for real-time scheduling

### Pre-Installed Packages
```
shoggoth-node-agent        # Auto-starts at boot, joins fabric via UDP broadcast.
shoggoth-cli               # CLI for local topology inspection and benchmarks.
nvidia-driver-570          # Latest stable NVIDIA driver with NVENC.
rocm-7.13                  # AMD ROCm for BC250 / MI50 compute.
docker-ce + nvidia-container-toolkit
tailscale                  # Mesh VPN for secure multi-site fabric.
prometheus-node-exporter   # Hardware metrics.
```

### Fabric Boot Sequence
1. Power on → UEFI → ShoggothOS boots.
2. `shoggoth-node-agent` systemd unit starts.
3. Node agent bootstraps local GPU fabric via wgpu.
4. Node agent broadcasts UDP heartbeat with hardware profile.
5. Orchestrator discovers node → registers in fabric pool.
6. Node is ready to receive WorkUnits within 15 seconds of boot.

## 3. Networking

### Ports
| Port | Protocol | Service |
|------|----------|---------|
| 9100 | QUIC/TCP | Orchestrator control plane |
| 9101 | WebSocket | Telemetry feed |
| 9102 | HTTP | Prometheus metrics |
| 8888 | UDP | Node heartbeat broadcast |
| 9150-9152 | Vsock | WSL2 DMA-BUF bridge |
| 9042 | TCP | ScyllaDB CQL |

### Mesh VPN
All Shoggoth Edge devices automatically join a Tailscale/WireGuard mesh:
- Pre-shared auth key baked into the OS image via cloud-init.
- Devices appear as `{serial}.shoggoth.ts.net`.
- Orchestrator resolves node IDs to Tailscale IPs for QUIC connections.

## 4. Cloud Integration

### Auto-Provisioning
When local edge nodes exceed 85% utilization:
1. Cloud Provisioner queries Brev.dev API for cheapest matching GPU.
2. Instance boots with the same ShoggothOS image.
3. Instance joins Tailscale mesh → appears as `cloud-{type}-{id}.shoggoth.ts.net`.
4. Node agent registers with orchestrator automatically.
5. When idle for 5+ minutes → auto-terminate.

### Supported Cloud Providers
- **Brev.dev**: A100 80GB ($1.10/hr), H100 80GB ($2.50/hr), L40S 48GB ($0.80/hr)
- **AWS**: P5 (H100 8×), G5 (A10G 4×)
- **GCP**: A3 (H100 8×)

## 5. SDK Onboarding

### For Developers
1. Install Shoggoth SDK: `pip install shoggoth-sdk` or `cargo add shoggoth-sdk`.
2. Create `shoggoth.toml` in project root (auto-generated by `shoggoth analyze`).
3. Run workload — the agentic parser routes it to optimal hardware automatically.
4. Monitor via Launchpad Dashboard or `shoggoth status`.

### For Hardware Integrators
1. Flash ShoggothOS image onto any x86_64 machine with a GPU.
2. Connect to same network as the orchestrator.
3. Power on — automatic fabric join.
4. Verify: `shoggoth topology` shows the new node.
